SnakeFolio

About

This project was developed as a part of the Web Programming 2 course, with the main objective of exploring the capabilities of the Three.js library. Initially, the intention was to create a simple page as a portfolio using Three.js and to experiment with this powerful tool. However, during the development process, an exciting idea emerged: to create a fun and engaging game using only geometric objects.

Instructions

Use the mouse to navigate and interact with the game. The mouse movement will control the direction of the snake's head. As you move the mouse, the snake's head will follow the movement. To move the snake just "Grab holding the right click", dont find the cube? What about zoom out or in completely. The purpuse is to see how long can you grow your snake. Be careful, you might see some projects along the way.
